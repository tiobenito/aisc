# Skill Builder

Build your first Claude Code skill in about an hour.

## What you'll build

A reusable skill -- a prompt that Claude loads automatically when you need it. Like a recipe card for a workflow you repeat often.

## How to use it

```bash
git clone https://github.com/tiobenito/aisc skill-builder
cd skill-builder/skill-builder
claude
```

Claude will walk you through everything from there.

## What's inside

- `CLAUDE.md` -- your interactive instructor (Claude reads this automatically)
- `examples/` -- 3 example skills to reference
