# Morning Briefing Agent

Build a Claude Code skill that reads your email and calendar and gives you a daily briefing.

## What you'll build

A `/briefing` skill that connects to Gmail and Google Calendar, summarizes your day, and highlights what needs attention.

## Prerequisites

- Completed the Skill Builder exercise (you know what a skill is)
- A Gmail account
- A Google Calendar

## How to use it

```bash
git clone https://github.com/tiobenito/aisc morning-briefing
cd morning-briefing/morning-briefing
claude
```

Claude will walk you through everything from there.

## What's inside

- `CLAUDE.md` -- your interactive instructor (Claude reads this automatically)
- `examples/` -- sample briefing output to see what you're building toward
- `skill-template/` -- starter template for your briefing skill
