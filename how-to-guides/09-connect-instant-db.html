Guide 09

How to Connect Instant DB to Your App

Set up a real-time database for your app using Instant.

Prerequisites

  • A working React/Vite project
  • A free Instant account (sign up at instantdb.com)
1

Create an Instant app

Go to instantdb.com, sign in, and click Create App. Give it a name.

Copy your App ID from the dashboard. You'll need it in a moment.

⬚ Screenshot or screen recording

Instant dashboard showing the App ID location

2

Install the SDK

In your project folder:

zsh
$ npm install @instantdb/react
3

Initialize Instant in your app

Create a file called src/db.ts:

src/db.ts
import { init } from '@instantdb/react'; const APP_ID = 'YOUR_APP_ID_HERE'; const db = init({ appID: APP_ID }); export default db;

Replace YOUR_APP_ID_HERE with the App ID you copied.

Tip

For production apps, put the App ID in an environment variable instead of hardcoding it.

4

Read data

In any React component:

RecipeList.tsx
import db from '../db'; function RecipeList() { const { isLoading, error, data } = db.useQuery({ recipes: {} }); if (isLoading) return <div>Loading...</div>; if (error) return <div>Error: {error.message}</div>; return ( <ul> {data.recipes.map((recipe) => ( <li key={recipe.id}>{recipe.title}</li> ))} </ul> ); }
⬚ Screenshot or screen recording

Browser showing a list of items pulled from Instant DB

5

Write data

To add a new item:

addRecipe.ts
import { id } from '@instantdb/react'; import db from '../db'; function addRecipe(title: string) { db.transact(db.tx.recipes[id()].update({ title, createdAt: Date.now() })); }

The id() function generates a unique ID. transact sends the change to the database.

6

Verify in the dashboard

Go back to instantdb.com, open your app, and click Explorer. You should see your data there.

⬚ Screenshot or screen recording

Instant dashboard Explorer showing the data that was just written

Key Concepts

  • useQuery - reads data and updates in real-time (no refresh needed)
  • transact - writes data (create, update, delete)
  • id() - generates unique IDs for new records
  • Data syncs instantly between all connected clients

Troubleshooting

"App not found"

Double-check your App ID. Copy it fresh from the Instant dashboard.

Data doesn't show up

Check the Explorer in the Instant dashboard. If it's there but not in your app, check your useQuery syntax.

Type errors

Instant works fine without a schema, but for TypeScript projects, check the Instant docs for typed schema setup.
