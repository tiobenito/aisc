Guide 01

How to Install and Set Up Claude Code

Get Claude Code running on your machine in under 10 minutes.

Prerequisites

  • A computer with macOS, Windows, or Linux
  • An Anthropic account — sign up at console.anthropic.com
1

Install Node.js

Go to nodejs.org and download the LTS version. Run the installer.

Verify it worked:

zsh
$ node --version v20.11.0

You should see something like v20.x.x or higher.

⬚ Screenshot or screen recording

Terminal showing node --version output

2

Install Claude Code

zsh
$ npm install -g @anthropic-ai/claude-code
Tip

If you get a permissions error, try sudo npm install -g @anthropic-ai/claude-code and enter your password.

3

Start Claude Code

Navigate to any project folder — or just your Desktop to test:

zsh
$ cd ~/Desktop $ claude
4

Authenticate

The first time you run claude, it will open your browser to log in with your Anthropic account. Follow the prompts and come back to your terminal.

⬚ Screenshot or screen recording

Browser authentication flow and terminal waiting for login

5

Verify it works

Once authenticated, type something in the Claude Code prompt:

claude
What files are in this directory?

If Claude responds with a list of files, you're all set.

⬚ Screenshot or screen recording

Claude Code responding to a simple prompt in the terminal

Troubleshooting

"command not found: claude"

Close your terminal and open a new one. The install needs a fresh terminal session to register the new command.

"npm not found"

Node.js didn't install correctly. Re-download the LTS version from nodejs.org and run the installer again.

Authentication loop

Make sure you're logged into your Anthropic account in the browser that opens. Try a different browser if it keeps looping.
