# Presentation Builder

Turn rough notes into polished HTML slide decks. No PowerPoint, no design skills, no external tools.

## What you'll build

A `/make-deck` skill that takes messy notes and produces a clean, professional slide deck you can open in any browser.

## How to use it

```bash
git clone https://github.com/tiobenito/aisc
cd aisc/exercises/presentation-builder
claude
```

Claude will walk you through everything from there.

## What's inside

- `CLAUDE.md` -- your interactive instructor (Claude reads this automatically)
- `templates/` -- HTML slide deck template with professional styling
- `samples/` -- rough notes to practice with
- `examples/` -- a finished deck to see what the output looks like
- `skill-template/` -- starter template for the skill you'll build
