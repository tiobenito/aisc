# Data Cleanup & Report

Turn messy data into clean, structured files with summary reports -- in minutes instead of hours.

## What you'll build

A reusable `/clean-data` skill that takes any messy CSV (inconsistent dates, duplicate rows, typos, missing fields) and produces a cleaned version plus a summary report.

## How to use it

```bash
git clone https://github.com/tiobenito/aisc data-cleanup
cd data-cleanup/data-cleanup
claude
```

Claude will walk you through everything from there.

## What's inside

- `CLAUDE.md` -- your interactive instructor (Claude reads this automatically)
- `samples/` -- two messy datasets to practice on (expenses + contacts)
- `examples/` -- reference outputs showing what clean looks like
- `skill-template/` -- starter template for your `/clean-data` skill
