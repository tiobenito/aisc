# Meeting-to-Action Pipeline

Build a Claude Code skill that turns messy meeting transcripts into structured summaries with action items, owners, and deadlines.

## What you'll build

A reusable `/meeting-summary` skill that takes any meeting transcript or rough notes and outputs a clean summary formatted for Slack, email, or Notion -- with action items, owners, deadlines, and key decisions extracted automatically.

## How to use it

```bash
git clone https://github.com/tiobenito/aisc meeting-to-action
cd meeting-to-action/meeting-to-action
claude
```

Claude will walk you through everything from there.

## What's inside

- `CLAUDE.md` -- your interactive instructor (Claude reads this automatically)
- `samples/` -- 2 sample meeting transcripts to practice with
- `skill-template/` -- starter SKILL.md template
- `examples/` -- what the finished output looks like (Slack and email formats)
