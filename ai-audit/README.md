# AI Audit

Map out your work and personal life to find where AI can save you real time.

## What you'll build

A prioritized list of 15-20 tasks you can automate, delegate to AI, or build workflows for. This becomes your personal roadmap for what to build next.

## How to use it

```bash
git clone https://github.com/tiobenito/aisc ai-audit
cd ai-audit/ai-audit
claude
```

Claude will walk you through everything from there.

## What's inside

- `CLAUDE.md` -- your interactive instructor (Claude reads this automatically)
- `examples/` -- a sample completed audit for reference
